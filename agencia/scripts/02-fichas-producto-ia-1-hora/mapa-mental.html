n8n · IA · Shopify · Google Sheets

¿Por qué importa?

  • 87% de compradores decide por la descripción
  • Ficha mala → menos ventas + menos Google
  • Agentes IA leen fichas para recomendar productos
  • Sin tiempo y sin ser copywriter → problema doble

Qué genera el sistema

  • Título SEO · máx 70 caracteres
  • Meta description · 155 caracteres + CTA
  • Descripción · 150-200 palabras, tono de marca
  • 5 bullets de beneficios (no características)
  • Tags de Shopify sugeridos

Catálogo en Sheets

  • ID producto Shopify
  • Nombre + Categoría
  • Características técnicas
  • Público objetivo
  • Precio · Diferenciador clave
  • Columna "Generado" (vacío / ✓)

Flujo del workflow

📊 Google Sheetslee cada fila
→

→
🤖 Claude / GPT-4genera las 5 partes
→
🛍️ Shopify APIactualiza en directo
→
✅ Sheetsmarca como generado
