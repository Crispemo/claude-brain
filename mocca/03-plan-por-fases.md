# Mocca — Plan por Fases
> Objetivo: lanzar una marca de complementos para perros desde cero, sin presupuesto, con validación previa

---

## Visión general del plan

```
FASE 0 — Investigación     → saber dónde y cómo entrar
FASE 1 — Validación        → probar que hay demanda real antes de invertir
FASE 2 — Comunidad         → construir audiencia antes del lanzamiento
FASE 3 — Lanzamiento       → abrir ventas con lista de espera caliente
```

---

## FASE 0 — Investigación (AHORA — 2 semanas)

**Objetivo:** entender el mercado antes de comprometer ni un euro.

### Tarea 0.1 — Estudio de mercado con Claude
```
Actúa como market researcher especializado en ecommerce de productos para mascotas en España.
El producto es: complementos para perros (suplementos, accesorios, alimentación complementaria — definir).
Dime:
1. Tamaño y tendencias del mercado de mascotas en España
2. Tipos de productos con mayor demanda en el segmento complementos para perros
3. Precio medio y márgenes típicos del sector
4. Comportamiento del consumidor: dónde compra, qué valora, qué le frena
5. Estacionalidad (hay meses con más demanda)
```

### Tarea 0.2 — Análisis de competencia en Instagram y TikTok
- [ ] Identificar 10 marcas de complementos para perros activas en RRSS en España
- [ ] Para cada una: seguidores, formato de contenido, precio, propuesta de valor, engagement
- [ ] Identificar qué están haciendo bien y qué huecos de posicionamiento existen

**Prompt para Claude:**
```
Actúa como competitor analyst de marcas de complementos para perros en España.
Estas son las marcas que he encontrado: [listar]
Analiza sus patrones de contenido, posicionamiento y propuesta de valor.
Dime: qué están haciendo bien, qué huecos existen, y cómo debería posicionarse Mocca para diferenciarse.
```

### Tarea 0.3 — Definir el producto inicial
- [ ] Decidir qué tipo de complemento lanzar primero (1 producto, no 10)
- [ ] Criterios: margen alto, diferenciación posible, fácil de demostrar en vídeo, problema real
- [ ] Consultar con Claude comparativa de opciones

### Tarea 0.4 — Documento de posicionamiento de marca
Una vez tengamos los datos, crear `02-posicionamiento-marca.md` con:
- Propuesta de valor única
- Tono de comunicación
- Nombre + identidad visual (referencias, no definitivo)
- Para quién es y para quién no es

## FASE 1 — Validación (semanas 3-6)

**Objetivo:** conseguir X reservas o suscriptores a lista de espera ANTES de pedir stock.

### Tarea 1.1 — Crear la lista de espera
- [ ] Landing page simple (puede ser una página de Notion pública, Carrd o similar)
- [ ] Formulario de email o WhatsApp para lista de espera
- [ ] Propuesta de incentivo: precio especial para early adopters, acceso anticipado, regalo con primera compra

### Tarea 1.2 — Vídeos de validación (el motor)
- [ ] Claude escribe 5 guiones adaptados al estilo del usuario (ver `04-guiones-validacion.md`)
- [ ] Tipos de vídeos de validación:
  1. "Estoy lanzando algo para dueños de perros y necesito tu opinión" (genera curiosidad + feedback)
  2. "El problema que vi con [X] y por qué decidí crear Mocca" (historia + propósito)
  3. "¿Le darías esto a tu perro?" (pregunta directa + CTA a lista)
  4. "Primeras 100 personas que se apunten tendrán [beneficio]" (urgencia + conversión)
  5. Teaser del producto sin revelar todo (intriga)

### Tarea 1.3 — Métrica de validación
Definir el umbral antes de invertir en stock:
- [ ] X personas en lista de espera (definir número)
- [ ] X reservas con pago o señal
- [ ] X interacciones en vídeos de producto

## FASE 2 — Construcción de comunidad (semanas 4-10, paralela a validación)

**Objetivo:** que cuando llegue el día del lanzamiento, haya gente esperándolo.

### Tarea 2.1 — Estrategia de contenido orgánico
- [ ] Crear cuenta de Instagram y/o TikTok de Mocca
- [ ] Publicar contenido que eduque sobre el producto mientras genera deseo y pertenencia
- [ ] 3-4 publicaciones por semana

**Tipos de contenido de comunidad:**
- Contenido educativo: "Por qué los perros necesitan [X]", "3 señales de que tu perro necesita [suplemento]"
- Contenido emocional: fotos y vídeos de perros, historias de dueños, behind the scenes
- Contenido de validación: encuestas, preguntas, feedback de la lista de espera
- Contenido de marca: valores de Mocca, proceso de creación, por qué existe la marca

### Tarea 2.2 — Colaboraciones sin coste
- [ ] Identificar 10 cuentas de dueños de perros con entre 5k-50k seguidores (micro-influencers)
- [ ] Proponer colaboración a cambio de producto (cuando lo haya) o simplemente menciones cruzadas
- [ ] Participar activamente en comunidades de dueños de perros en Instagram, Facebook y grupos de WhatsApp

### Tarea 2.3 — Crear el calendario editorial con Claude
```
Actúa como content planner para una marca nueva de complementos para perros en España llamada Mocca.
Estamos en fase pre-lanzamiento, sin presupuesto para ads.
El objetivo del contenido es: educar sobre el producto, generar comunidad y conseguir suscriptores a la lista de espera.
Genera el calendario editorial para el primer mes: qué publicar cada día, en qué formato, con qué mensaje.
```

## FASE 3 — Lanzamiento (cuando se alcance la métrica de validación)

**Objetivo:** abrir ventas con una lista de espera caliente y convertirla en primeras ventas reales.

### Tarea 3.1 — Preparación del lanzamiento (1 semana antes)
- [ ] Definir fecha exacta de apertura del carrito
- [ ] Preparar tienda online (Shopify básico o similar)
- [ ] Configurar emails automáticos para la lista de espera
- [ ] Preparar stocks mínimos según reservas confirmadas

### Tarea 3.2 — Secuencia de lanzamiento (7 días)

| Día | Acción |
|---|---|
| D-7 | Email a lista: "Esto es lo que viene" (teaser) |
| D-5 | Vídeo en RRSS: revelación del producto |
| D-3 | Email: "En 3 días abrimos" + beneficios early adopter |
| D-1 | Story/Reel: cuenta atrás |
| D-0 | Email de apertura + post en RRSS + stories todo el día |
| D+1 | Primeras compras / testimonios de primeros compradores |
| D+3 | Recordatorio de cierre de oferta early adopter |
| D+7 | Cierre de oferta especial, precio sube |

### Tarea 3.3 — Guiones y copies del lanzamiento
Claude puede escribir todos los textos:
- Emails de la secuencia de lanzamiento
- Copies para posts e historias
- Textos de la landing/tienda
- Respuestas a preguntas frecuentes

```
Actúa como launch planner y copywriter para el lanzamiento de Mocca, marca de complementos para perros.
Tenemos X personas en lista de espera. El producto es [X]. El precio es [X].
Escribe la secuencia de 7 emails de lanzamiento con: asunto, preheader y cuerpo del email.
Tono: cercano, auténtico, no agresivo. El comprador es un dueño de perro que cuida mucho a su mascota.
```
